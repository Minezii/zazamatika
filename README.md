# ZazaMatika - Шахматы с суперспособностями

**ZazaMatika** — это современное веб-приложение для игры в шахматы, сочетающее классические правила с уникальными игровыми механиками (система аппетита и боевого духа) и возможностью сетевой игры в реальном времени.

![ZazaMatika UI](https://api.dicebear.com/7.x/avataaars/svg?seed=Felix&backgroundColor=b6e3f4) *(Сгенерировано для демонстрации)*

---

## ✨ Основные особенности

- **Сетевая игра**: Создавайте комнаты и играйте с друзьями по уникальному коду через Socket.io.
- **Уникальные способности**: Каждая фигура обладает специальной активной способностью (энергия накапливается через систему аппетита).
- **Система Аппетита и Боевого духа**: Фигуры "голодают" при движении и восстанавливают силы при взятии противника.
- **Самоуничтожение**: Фигуры могут взорваться, нанося урон окружающим клеткам.
- **История ходов и отмена**: Полная история партии с возможностью навигации и запроса отмены хода.
- **Современный дизайн**: Интерфейс в стиле Glassmorphism с приятными анимациями и адаптивностью под мобильные устройства.

### 🛡️ Способности фигур

- **Пешка**: Мгновенное превращение в Слона.
- **Конь**: Телепортация на любую свободную клетку.
- **Слон**: Рентген-взятие (сквозь другие фигуры).
- **Ладья**: Захват до 3 фигур по линии за один ход.
- **Ферзь**: Бессмертие на 3 следующих хода.
- **Король**: Призыв верного Ферзя.

## 🚀 Технологический стек

### Frontend
- **HTML5 & CSS3**: Кастомные стили с использованием CSS переменных и Glassmorphism.
- **JavaScript (ES6+)**: Чистая логика без тяжелых фреймворков.
- **Chess.js**: Основная логика шахматных правил.
- **Socket.io Client**: Реальное время для мультиплеера.

### Backend
- **Node.js**: Среда выполнения.
- **Express**: Минималистичный веб-сервер.
- **Socket.io**: Серверная часть WebSocket для управления игровыми комнатами.

## 🛠️ Установка и запуск

1. **Клонируйте репозиторий**:
   ```bash
   git clone <url_репозитория>
   cd chess-app
   ```

2. **Установите зависимости**:
   ```bash
   npm install
   ```

3. **Запустите сервер**:
   ```bash
   node server.js
   ```

4. **Откройте приложение**:
   Перейдите по адресу `http://localhost:3000` в вашем браузере.

---

## 🎮 Как играть

1. Выберите режим "Игра по сети" или просто играйте локально.
2. Нажмите на фигуру, чтобы увидеть доступные ходы и её текущие характеристики (Сытость/Боевой дух) на панели слева.
3. Когда шкала аппетита достаточно заполнена, станет доступна кнопка **⚡ Активировать способнось**.
4. Используйте кнопку **💥 Самоуничтожение** в критических ситуациях для тактического преимущества.
